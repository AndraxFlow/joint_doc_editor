# DocCollab — Web-приложение для совместного редактирования документов

## Быстрый старт

1. Клонировать репозиторий  
2. Создать файл `.env` на основе `.env.example`  
3. Запустить окружение:
# docker-compose up --build

4. Открыть приложение:  
http://localhost:8000

## Стек
- Python 3.11
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Docker
